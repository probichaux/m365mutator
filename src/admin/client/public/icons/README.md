# Workload product icons

Used on the Deletions page to label each workload.

- `outlook.svg`, `onedrive.svg` — official Microsoft product icons, from Microsoft's
  Office UI Fabric brand-icon CDN
  (`static2.sharepointonline.com/files/fabric/assets/brand-icons/product/svg/`).
- `outlook-calendar.svg` — a custom Fluent-style calendar glyph matching the family's
  blue palette. Microsoft publishes no standalone Calendar product icon (Calendar is
  part of Outlook), so this one is hand-authored.

Outlook and OneDrive are trademarks of Microsoft Corporation, used here only to
identify the corresponding workloads.
