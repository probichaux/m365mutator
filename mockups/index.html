M365Mutator — Run-style mockups

Explicit vs. Random run style

Three directions for letting the user choose, on every workload tab, between running on an explicit edited list or on a random percentage loaded from the tenant. Open each and toggle between the two modes.

  • 1 — Segmented switchA compact Explicit / Random toggle above the list; the body swaps to a percentage slider with a live count.
  • 2 — Choice cardsTwo descriptive radio cards explain each mode; the selected one reveals its controls (a +/− stepper for the percentage).
  • 3 — Header dropdownA small Run-style dropdown in the panel header; Random shows the tenant pool with an inline percentage and a live selection line.
